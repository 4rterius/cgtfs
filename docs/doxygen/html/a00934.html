◆ transfer_type_t

enum transfer_type_t

#include <includes/records/transfers.h>

Whether a transfer between two stops is possible and how.

Enumerator
TT_RECOMMENDED 

This is a recommended transfer point between routes (also on empty field value).

TT_TIMED 

Departing vehicle waits for the arriving one, with enough time for a passenger to transfer.

TT_TIME_REQUIRED 

This transfer requires a minimum amount of time between arrival and departure to ensure a connection.

TT_NOT_POSSIBLE 

Transfers are not possible between the specified stops.

TT_NOT_SET
