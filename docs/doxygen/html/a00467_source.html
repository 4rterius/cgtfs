1 # CGTFS - a C library to read static GTFS feeds {#mainpage}
2 
3 A thin and fast low-level library which reads GTFS static feeds. This library provides a readable and intuitive C interface for parsing data provided in the Google's [General Transit Feed Specification](https://developers.google.com/transit/gtfs/) format.
4

27 ## Build process and dependencies
28 
29 ### Dependencies
30 
31 One of the development goals was to keep the dependencies as minimal as possible. So far, CGTFS has following dependencies:
32 
33  - [greatest](https://github.com/silentbicycle/greatest) for tests
34  - [sqlite3](https://www.sqlite.org/index.html) for fast & efficient querying *built through [a special cmake-friendly repo](https://github.com/rakhack/sqlite3-cmake)*
35 
36 ### Build process
37 
38 The library should compile on `gcc >= 4.8.4`, `clang >= 5.0.0` and latest Microsoft's C/C++ compiler.
39 
40 Configuration: `Release` or `Debug`.
41 
42 #### Linux
43 
44 ```
45 $ cd /path/to/cgtfs/
46 $ git submodule update --init --recursive
47 $ mkdir build && cd build/
48 $ cmake -DCMAKE_BUILD_TYPE=%Configuration% ..
49 $ cmake --build .
50 
51 $ ./tests
52 ```
53 
54 #### Windows
55 
56 ```
57 cd \path\to\cgtfs\
58 git submodule update --init --recursive
59 mkdir build && cd build
60 cmake ..
61 cmake --build . --config %Configuration%
62 
63 %Configuration%\tests.exe
64 ```
