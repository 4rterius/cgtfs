1 # CGTFS - a C library to read static GTFS feeds {#mainpage}
2 
3 A thin and fast low-level library which reads GTFS static feeds. This library provides a readable and intuitive C interface for parsing data provided in the Google's [General Transit Feed Specification](https://developers.google.com/transit/gtfs/) format.
4 
5 ## Build process and dependencies
6 
7 ### Dependencies
8 
9 One of the development goals was to keep the dependencies as minimal as possible. So far, CGTFS has following dependencies:
10 
11  - [greatest](https://github.com/silentbicycle/greatest) for tests
12  - [sqlite3](https://www.sqlite.org/index.html) for fast & efficient querying *built through [a special cmake-friendly repo](https://github.com/rakhack/sqlite3-cmake)*
13 
14 ### Build process
15 
16 The library should compile on `gcc >= 4.8.4`, `clang >= 5.0.0` and latest Microsoft's C/C++ compiler.
17 
18 Configuration: `Release` or `Debug`.
19 
20 #### Linux
21 
22 ```
23 $ cd /path/to/cgtfs/
24 $ git submodule update --init --recursive
25 $ mkdir build && cd build/
26 $ cmake -DCMAKE_BUILD_TYPE=%Configuration% ..
27 $ cmake --build .
28 
29 $ ./tests
30 ```
31 
32 #### Windows
33 
34 ```
35 cd \path\to\cgtfs\
36 git submodule update --init --recursive
37 mkdir build && cd build
38 cmake ..
39 cmake --build . --config %Configuration%
40 
41 %Configuration%\tests.exe
42 ```
