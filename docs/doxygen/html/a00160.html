Detailed Description

Functionality related to each particular entity.

For each type of entity, CGTFS provides the following:

  • typedef'ed structure (T);
  • init_T(T *) function which initializes the struct's fields with default values (strings with "", numerics with 0, enumerations with default values);
  • read_T(T *, int, const char **, const char**) function which can fill the structure from an array of field names and field values (provided in the same order);
  • typedef'ed enums for field enumerations (when present), and functions for parsing the enumeration value from c-strings.
